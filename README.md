# Cowsay package

This is a small wrapper for cowsay for Atom. Just a small fun package. It takes the text you have selected and prints a cow saying your text.

## Usage

Just select a text and invoke cowsay command or press `crtl-l`.

```
 _____________
< Have fun!!! >
 -------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||

```
